# Built with
- Next.js
- React
- Tailwind
- [AOS](https://michalsnik.github.io/aos/) 

# Run it on your local computer

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
